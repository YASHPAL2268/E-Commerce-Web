# 🛒 E-Commerce Web (Frontend)

This is the **frontend** portion of a full-fledged E-Commerce Web Application built using the **MERN stack**:

- **MongoDB** – Database
- **Express.js** – Backend framework
- **React.js** – Frontend library
- **Node.js** – Runtime environment

> ⚠️ This repository contains **only the React.js frontend**. Backend APIs are expected to be connected to a separate Node/Express server.

---

## 🚀 Features

- 🔍 Product browsing and filtering
- 🛍️ Add to cart & cart management
- 🧑 User authentication (login/signup)
- 📦 Checkout flow (connected to backend)
- 🎨 Responsive UI (fully mobile-friendly)

---

## 🛠️ Tech Stack

| Tech            | Used For             |
|-----------------|----------------------|
| React.js        | Frontend framework   |
| React Router    | Client-side routing  |
| Axios / Fetch   | API requests         |
| Bootstrap / CSS | Styling and layout   |
